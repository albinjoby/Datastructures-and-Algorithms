# Data Structures and Algorithms

Welcome to the **Data Structures and Algorithms** repository! This repository contains implementations of various data structures and algorithms in Python. It is designed to help you understand key concepts and improve your problem-solving skills.

## Contents

The repository is divided into the following sections:

### 1. Data Structures
- Stack
- Queue
- Linked List (Singly, Doubly, Circular)
- Hash Table
- Binary Search Tree
- Heap
- Graph (Adjacency List, Adjacency Matrix)

### 2. Algorithms
- **Sorting Algorithms:**
  - Bubble Sort
  - Quick Sort
  - Merge Sort
  - Insertion Sort
  - Selection Sort

- **Searching Algorithms:**
  - Binary Search
  - Linear Search

- **Graph Algorithms:**
  - Breadth-First Search (BFS)
  - Depth-First Search (DFS)
  - Dijkstra’s Algorithm
  - Kruskal’s Algorithm
  - Prim’s Algorithm

- **Other Algorithms:**
  - Dynamic Programming
  - Greedy Algorithms
  - Divide and Conquer
